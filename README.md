Project ini merupakan implementasi technical test Backend Developer dengan studi kasus pengelolaan hewan peliharaan milik Esa. Aplikasi ini dibangun menggunakan bahasa PHP dengan framework Laravel. Seluruh kebutuhan logika yang diminta pada soal diimplementasikan menggunakan pendekatan berbasis array tanpa database.

Cara Menjalankan Project
1.Clone repository.
2.Jalankan composer install.
3.Salin file .env.example menjadi .env.
4.Jalankan php artisan key:generate.
5.Jalankan server dengan php artisan serve.
6.Akses aplikasi melalui browser.
